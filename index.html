Calendario caraíta astronómico reforzado
Sistema reforzado para navegador usando Astronomy Engine para posiciones astronómicas, SunCalc para contraste solar/lunar y Luxon para zona horaria y formato. Cada evento se muestra con inicio y fin visibles.
Este modo usa varias capas: astronomía precisa, contraste solar/lunar y control explícito de zona horaria. La ancla opcional del mes 7 sirve para comparar el resultado o fijarlo manualmente si ya tienes el dato oficial.
Reglas astronómicas aplicadas
  • Se busca una luna nueva astronómica cercana al inicio del año.
  • Se revisan los atardeceres posteriores y se calcula una visibilidad estimada del creciente.
  • La decisión de inicio de mes usa: altura lunar, elongación Sol-Luna, retraso lunar, iluminación y la hora de puesta del sol.
  • Astronomy Engine aporta posiciones topocéntricas y luna nueva; SunCalc se usa para contraste de atardecer y tiempos lunares; Luxon controla interpretación y salida en una zona horaria definida.
  • Omer Reshit comienza al terminar el shabbat semanal dentro de Jag HaMatzot.
  • Shavuot se calcula como el día 50 desde Omer Reshit.
